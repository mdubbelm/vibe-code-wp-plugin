Plugin ideas

Not sure what to build? Pick an idea from this list and write your own prompt.

Use the prompt template from the preparation page to describe your chosen idea. Fill in the name, what it should do, where it should appear, and how it should behave — then paste it into your AI tool.

Frontend — visible to visitors

  • Last Updated Date Easy Shows "Last updated on [date]" above or below a blog post.
  • Copyright Footer Easy Adds "© [year] [site name]" to the bottom of every page. The year updates automatically.
  • Author Bio Box Easy Shows a short author description with avatar below each post.
  • Simple Cookie Notice Easy Shows a "This site uses cookies" banner at the bottom of the page with a dismiss button.
  • Custom 404 Message Easy Replaces the standard "Not Found" message with your own text and a link back to the homepage.
  • Reading Progress Bar Medium Shows a progress bar at the top of the page that fills as you scroll through a post.

Admin — visible in the WordPress dashboard

  • Admin Footer Text Easy Replaces "Thank you for creating with WordPress" at the bottom of every admin page with your own message.
  • Dashboard Welcome Widget Easy Adds a custom welcome message or quick-links panel to the WordPress dashboard.
  • Word Count Column Easy Adds a "Words" column to the posts overview so you can see the word count of each post at a glance.
  • Login Page Message Easy Adds a custom message below the login form — useful for multi-author sites ("Editors only — contact admin for access").
  • Missing Featured Image Warning Medium Shows a warning when you try to publish a post without a featured image set.

Site-wide — behind the scenes

  • Disable Emoji Easy Removes the WordPress emoji scripts from every page. Makes your site load slightly faster.
  • Remove Generator Tag Easy Removes the WordPress version number from your page source code. Small privacy and security improvement.
